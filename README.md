最新版本以maven仓库中版本为准

## 依赖

```xml

<dependency>
    <groupId>cn.enaium</groupId>
    <artifactId>kook-spring-boot-starter</artifactId>
    <version>最新版本</version>
</dependency>
```

```groovy
implementation 'cn.enaium:kook-spring-boot-starter:最新版本'
```

```kts
implementation("cn.enaium:kook-spring-boot-starter:最新版本")
```
